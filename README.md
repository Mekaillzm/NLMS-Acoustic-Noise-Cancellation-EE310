Download clean_audio.flac, AlgoNLMS.py, synth1.py and both of the graph files in the same folder.
Run the graph files to obtain desired graphs

clean_audio.flac offers a 29 second audio clip from an audio book, sourced from an open source library. This represents the far-end audio, making for a better test case under realistic conditions.

Alternatively, a simple sine-wave representation is also available for testing.

Both tests will return unacceptable ERLE values as the code has yet to be properly fine-tuned.
